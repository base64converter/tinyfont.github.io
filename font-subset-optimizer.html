Frequently Asked Questions

Q: Why should I subset my fonts?
A: Subsetting fonts drastically reduces their file size, leading to faster website loading times and improved performance, especially for users on slower connections.

Q: Does this tool provide the actual font files?
A: No, this tool generates the unique character string needed for subsetting. You will use this string with a dedicated font subsetting tool (like Google Webfonts Helper) to get the optimized font files.

Q: Can I use this with any font?
A: This tool is designed for Google Fonts. The generated character string can be used with any font subsetting tool that supports custom character ranges.
